# Digital Clock

A simple Digital clock built with HTML, CSS and JavaScript.
It displays the current time and includes features like Dark/Light theme toggle.

## Features
- Real-time digital clock display
- Dark/Light theme toggle
- Easy to extend with new features (Stopwatch, Countdown, Alarm)
